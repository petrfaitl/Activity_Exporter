Activity Exporter

Utility to convert sportswatch workout files to gpx format

About this project

This project had started as a response a Christmas 2014 outage of the Movescount website. This is a Suunto owned and operated site that is used to support Suunto watches and devices. This outage meant that users were unable to upload workouts from their watch to analyse and share with others.

YOUR SPORTS DIARY

Store every single workout you do with the amount of detail you want. Whether you are a competing athlete, an outdoor maniac, a technical diver or just getting started with your new active life - Movescount is for you!

As you require an active connection to this website to export your workout logs from your watch, it became necessary to design a tool to analyse and convert Suunto workout files to a standard recognised format, such as .gpx. These can be used at other online analytics services, for example Strava. This is where the Activity Exporter app comes into use.

Install and run

  • If using Suunto watch, sync your watch via the provided MovesCount app to extract workout files to your computer.
  • Download the ActivityExporter app .zip file from the download page.
  • Unzip the file to a location, where you'll find it afterwards (such as the Downloads folder). The file contains a .jar file that is the Exporter app. You'll require Java 7 on your system. You can verify which version you have and update or download java at the java.com website.
  • Run the Activity_Exporter.jar file to launch the app.
